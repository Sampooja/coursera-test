introduction to Rubik’s Mini

1) FACES

Each flat surface is called a face.

There are 6 faces on a Rubik’s Mini.

The UP face is on the Top Layer, The DOWN face is on the Bottom Layer.


2) When correctly solved

WHITE opposite YELLOW

BLUE opposite GREEN

ORANGE opposite RED


3) MOVES IN RUBIK’S MINI

F = FRONT FACE

F' = FRONT FACE INVERSE

B = BACK FACE

B' = BACK FACE INVERSE

U = UP FACE

U' = UP FACE INVERSE

D = DOWN FACE

D' = DOWN FACE INVERSE

R = RIGHT FACE

R' = RIGHT FACE INVERSE

L = LEFT FACE

L' = LEFT FACE INVERSE


4) Each move is a ¼ TURN.

5) An ALGORITHM is a sequence of moves that you need to do in a specific order.

Use the control buttton's given below.

6) When following the algorithms in this guide, it is important to maintain the FRONT face of your Rubik’s Mini so it stays the FRONT through all of the turns.

7) If there is a 2 next to the algorithm letter, turn the face twice.
